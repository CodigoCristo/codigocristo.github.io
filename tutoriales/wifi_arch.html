Conexión a Internet por Wifi

Si desea conectarse por wifi, es conveniente primero verificar si se ha cargado correctamente el controlador de la tarjeta wifi. Compruebe que en la salida del siguiente comando exista un fichero que inicie con la letra wl...

wlan0
wlp2s0
Otros...
[root@nombredepc ~]# ls /sys/class/net | grep wl

Si no existe ningún fichero que inicie con la letra wl ,entonces no podrá realizar la instalación vía wifi.
Si por el contrario el fichero existe entonces será posible conectarse a Internet vía wifi.


Conexión de Wifi con NMCLI o IWD

NMCLI

NMCLI es gracias a NetworkManager, Fuente: https://wiki.archlinux.org/title/NetworkManager#nmcli_examples

Entramos como root y Activamos el servicio de NetworkManager :

[root@nombredepc ~]# systemctl start --now NetworkManager

Ahora si podemos usar nmcli :

[root@nombredepc ~]# nmcli dev wifi list


Si la red tiene espacios solo ponemos comilla simple o doble comillas en el bssid:

[root@nombredepc ~]# nmcli device wifi connect 'NombreDeRed' password 'contraseña'

Verificamos la conexión:

[root@nombredepc ~]# nmcli dev status



IWD

IWD nos brinda el programa iwctl Fuente: https://wiki.archlinux.org/title/iwd
Para saber el nombre de device debemos revisar en ls /sys/class/net
Asi obtenemos el nombre de nuestro dispositivo de red

[root@nombredepc ~]# ls /sys/class/net | grep w

En mi caso fue wlo1



Activamos el servicio de IWD para poder usuarlo :

[root@nombredepc ~]# systemctl start --now iwd

Ahora si podemos usar IWD :

[root@nombredepc ~]# iwctl --passphrase 'contraseña' station wlo1 connect 'NombreDeRed'


Antes de hacer alguna instalación siempre actualizamos el sistema con:

[root@nombredepc ~]# pacman -Syu
